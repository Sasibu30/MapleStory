사용법 (펼치기)

- 사용 방법 -

1. 직업 및 직업 계열, 무기 종류 선택

2. 능력치 및 아이템에 붙은 공격력/마력 입력

3. 버프 스킬 선택 ( 선택 하지 않아도 계산 가능 )

4. '계산하기' 클릭 ( 마법사의 경우 스킬 선택 후 계산하기 클릭)

- 입력 정보 저장 방법-

5. '저장하기' 클릭할 경우 입력한 데이터가 저장

6. '불러오기' 클릭 후 'xxx.sasibu' 파일을 선택하여 입력값 불러오고 수정 및 계산 가능
